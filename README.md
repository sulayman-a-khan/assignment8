## 📖 Description

SkillSphere is an interactive and visually striking online learning platform designed to provide a seamless educational experience. It features robust route protection, secure authentication, and intuitive course search and filtering capabilities. The platform boasts a highly responsive, premium UI enhanced with modern design aesthetics to maximize student engagement.

## ✨ Key Features

- **🔐 Secure Authentication:** Seamless login and registration with Email and Google OAuth integrations.
- **🛡️ Protected Routes:** Robust middleware-based route protection to ensure privacy for authenticated users.
- **🔍 Advanced Search & Filtering:** Quickly find exactly what you want to learn with optimized course discovery tools.
- **🎨 Modern UI/UX:** A stunning, premium design utilizing Tailwind CSS and DaisyUI, featuring dynamic micro-animations and a sleek layout.
- **📱 Fully Responsive:** Carefully optimized to provide a flawless experience across all devices, from mobile phones to large desktop screens.
